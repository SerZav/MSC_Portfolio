UNIT 7 - Introduction to Operating Systems

Exploring a simple Python shell

We were required to create a Python shell that implements basic commands such as LIST, ADD, HELP and EXIT.

My first approach was to implement the Praka (2018) approach, but after reviewing the Szabo (2018) suggestion, it became apparent for me that the cmd library was best suited for readability and efficiency. Praka's approach could be improved by using a dictionary instead of if statements to achieve the same efficiency.

I never programmed a shell before, and I think it was a very interesting and important task. Being so busy with two modules and a full-time job, I spent a little more time in this exercise than I probably should, but at the same time, I wished I had more time to make a more complete and more functional shell. From the point of view of security, I feel that this was a subject that needed more time for deeper research. I will sure continue to develop these skills in the future.

As for operating systems and virtualization, I missed some content and hands-on exercises with containers and other related concepts. I understand that the shell itself was a good exercise that consumed some time, so maybe I would have appreciated these two concepts in separate weeks to have a better insight.
