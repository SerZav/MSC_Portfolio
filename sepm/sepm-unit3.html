One of the online applications I use on a daily basis is Google Maps. It was also the most downloaded map and navigation application in the United States in 2021 (Ceci L., 2022). Mapping applications such as Google Maps, are backed by a complex search system based on graphs.

Traversing graphs using pathfinding algorithms is a very efficient way to find the shortest path between two points. Google maps use every point through the route as a node and edges that connect every node. Each node has a weight depending on several factors such as distance between them, traffic, allowed speed, and so on. The algorithm will try to find the route with the less sum of weights to determine the shortest path, or fastest route to reach a destination.
